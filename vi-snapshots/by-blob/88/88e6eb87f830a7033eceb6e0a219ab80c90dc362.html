ServerListener.lvlib:ServerListener.lvclass:Lookup or Assign Name of Client Using GUID.vi

Given the GUID, this method will check to see if this client has previously connected. If it has, it will return the name that was previously used. If it has not, it will assign a new one and store the GUID in a lookup table.
